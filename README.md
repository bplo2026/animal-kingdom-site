# Animal Kingdom – strona statyczna

Prosta strona statyczna pod GitHub Pages.

## Pliki

- `index.html` – strona główna
- `kontakt.html` – kontakt
- `regulamin.html` – placeholder regulaminu
- `polityka-prywatnosci.html` – placeholder polityki prywatności
- `assets/style.css` – style strony
- `CNAME` – domena dla GitHub Pages: animal-kingdom.pl

## Publikacja na GitHub Pages

1. Utwórz repozytorium na GitHubie.
2. Wrzuć wszystkie pliki z tego folderu do głównego katalogu repozytorium.
3. Wejdź w Settings → Pages.
4. Ustaw Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. W Custom domain wpisz: animal-kingdom.pl.
7. Włącz Enforce HTTPS, gdy będzie dostępne.

## Do podmiany przed finalną publikacją

- adres e-mail, jeśli ma być inny niż kontakt@animal-kingdom.pl
- treść regulaminu
- treść polityki prywatności
- dane właściciela / administratora, jeśli mają być publicznie podane
